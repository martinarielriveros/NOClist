# NOC List

Create a series of tasks using clean arquitecture with typescript

## dev

1. Create an `.env`
2. Configure evvironment variables like:

```Typescript
    PORT=
    MAILER_EMAIL=
    MAILER_PASSWORD=
    MAILER_HOST=
    MAILER_PORT=
    MAILER_SECURE=
```

3. Run `npm install`
4. Run `npm run dev`
